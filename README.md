## Getting Started
1. Clone the repo and install dependencies in both `client` and `server` folders:
   ```sh
   cd client
   npm install
   cd ../server
   npm install
   ```
2. update your .env file
   
3. Start the backend:
   ```sh
   npm start
   ```
4. Start the frontend:
   ```sh
   cd ../client
   npm start
   ```
